Important FAQ

What is the Document Object Model (DOM)?

DOM stand for Document Object Model. dom is a special type of object model. with the help of dom javascript can make any changes on html element like modify, add, delete etc. without dom javascript can't make any changes on html.

How do you select an element from the DOM?

we have few method to select element from dom

  • getElementById
  • getElementByClassName
  • getElementByTagName
  • querySelector
  • querySelectorAll
What is event delegation in the context of the DOM, and why is it useful?

event deligation is deligate event to the parent. it's useful because if we manually add event listener and we got dynamic element then manually added listener won't work. here comes deligation into play. it will add listener to parent then no matter what data is static or dynamic it will work all time.

How do you manipulate an element's attributes and styles using the DOM?

we have few method to manipulate them. like:

setAttribute to set an attribute

remove attribute to remove an attribute

and we have style property . with that we can manipulate styles also

we can add class with method also that way we can manipulate tailwind style also
